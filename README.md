## 🌊 关于 Cerulena

> *"Cerulean" — 源自拉丁语 *caeruleus*，意为天蓝色、海洋的颜色。*

这是一个基于 GitHub Pages 搭建的个人网站，色调灵感来自深邃而宁静的天蓝色 `#007ba7`。

### ✨ 特点

- 🎨 **极简设计** — 以 `#007ba7` 为主色调，搭配清爽的视觉风格
- 📱 **响应式布局** — 适配各种设备屏幕
- ⚡ **静态部署** — 利用 GitHub Pages 快速加载
